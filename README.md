#   Bulk Email Sender

**Bağımlılıklar:**

-   Ruby 1.9.3
-   roo, tlsmail, gtk2 gemleri

##  Linux

### Kurulum && Kullanım


-   Gem Kurulumları:

```sh
$ sudo gem install roo tlsmail gtk2

```

-   Uygulamanın Kurulumu && Kullanımı

```sh
$ git clone git://github.com/ecylmz/bulkemail-gui.git
$ cd bulkemail-gui
$ ruby bulkemail.rb
```
